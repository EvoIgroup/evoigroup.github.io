The Evolutionary Intelligence (EvoI) group was founded in January 2023, affiliated with Xidian University (XDU), China. Generally, our research motivations can be interpreted at four levels:

  • EvoI for Network Sciences
    • Hypergraphs
    • Causal Inference
    • Community Detection
    • Higher Order Networks
    • Influence Maximization
    • Network Reconstruction
  • EvoI for Machine Learning
    • Auto Learning
    • Meta Learning
    • Kinetic Analysis
    • Graph Learning
    • Generative Model
    • Adversarial Learning
  • EvoI for Optimization
    • Numerical Optmization
    • Expensive Optimization
    • Large-Scale Optimization
    • Combinatorial Optimization
    • Multi-objective Optimization
    • Multi-task/Transfer Optimization
  • EvoI for Application
    • Cyber security
    • Time Series Analysis
    • Black-box Applications
    • Placement and Routing
    • Bioinformatics and Life Sciences
